# Dotfiles Project Context

## Project Overview
Personal configuration and environment bootstrapper for macOS using GNU Stow and Homebrew.

## Key Commands
- **Setup:** `./setup.sh` - Installs all tools, creates symlinks, installs Claude Code
- **Claude Code install:** `npm install -g @anthropic-ai/claude-code`
- **API key setup:** `echo 'export DEEPSEEK_API_KEY="your-key"' >> ~/.api-keys`
- **Zsh config:** `zshconfig` (alias for `code ~/Projects/dotfiles/zsh/.zshrc`)
- **Navigate to dotfiles:** `dotfiles` (alias for `cd ~/Projects/dotfiles`)
- **Customize prompt:** `p10k configure`

## Project Structure
- Uses GNU Stow for symlink management: `stow -t ~ zsh git tmux bat fzf claude-config`
- Packages: `zsh/`, `git/`, `tmux/`, `bat/`, `fzf/`, `claude-config/`
- Key files: `Brewfile`, `setup.sh`, `README.md`

## AI CLI Configuration
- Claude Code configured to use DeepSeek API via Anthropic-compatible endpoint
- Environment variables in `.zshrc`:
  - `ANTHROPIC_AUTH_TOKEN=$DEEPSEEK_API_KEY`
  - `ANTHROPIC_BASE_URL=https://api.deepseek.com/anthropic`
  - `ANTHROPIC_MODEL=deepseek-reasoner`
  - `ANTHROPIC_DEFAULT_HAIKU_MODEL=deepseek-chat`
  - `ANTHROPIC_API_KEY` is unset to prevent conflicts
- API keys loaded from `~/.api-keys` (excluded from git)

## Common Aliases
- `ls='eza --icons'`
- `cat='bat'`
- `find='fd'`
- `grep='rg'`

## Git Aliases
- `st=status`
- `co=checkout`
- `br=branch`
- `hist=log --pretty=format:'%h %ad | %s%d [%an]' --graph --date=short`

## Tmux Commands
- New session: `tmux new -s <name>`
- List sessions: `tmux ls`
- Attach: `tmux a -t <name>`

## Important Notes
- `~/.api-keys` is excluded from version control
- Claude Code automatically installed by `./setup.sh` if not present
- Node.js installed via Homebrew for npm
- Restart terminal after setup for changes to take effect
